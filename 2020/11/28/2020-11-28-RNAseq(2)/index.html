# 1.2 Reference Genomes

从 Ensembl、iGenomes、NCBI 或 UCSC 获得参考基因组。在本例分析中，我们将使用人 GRCh38 版本的 Ensembl 基因组。此外，我们实际上将只使用单个染色体 (chr22) 和 ERCC spikein 来执行分析，以使它运行得更快……
